# 🎨 Анализатор цветовой палитры изображений

Простая Python-программа для анализа доминирующих цветов на изображении с использованием OpenCV.

## 📋 Описание

Программа загружает изображение, анализирует его пиксели в цветовом пространстве HSV и выводит статистику по цветовой палитре — какие цвета преобладают и в каком процентном соотношении.

## 🚀 Возможности

- Определение 11 базовых цветов: Красный, Оранжевый, Желтый, Зеленый, Бирюзовый, Синий, Фиолетовый, Розовый, Белый, Черный, Серый
- Процентное соотношение каждого цвета на изображении
- Визуальная шкала для наглядности
- Автоматическое масштабирование больших изображений для ускорения обработки
- Просмотр изображения в окне OpenCV
